This is a mod all about creating castles and castle dungeons. Many of the nodes are used for the outer-walls or dungeons.

=-=-=-=-=-=-=-=-=-=

Contains as of now:

--Walls, Corner-walls,
--Castlestone Stairs, Slabs, and Pillars
--Jailbars
--Hides
--Arrows and Arrowslits
--Rubble
--Doors
--Weapons
--Chandeliers
--Tapestries
--and more!
